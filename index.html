Cara Simpan ke GitHub

Muat turun fail assets.json, kemudian commit ke repo GitHub anda di data/assets.json. Aplikasi ini juga boleh dihost sebagai GitHub Pages sepenuhnya percuma.

Pilih Format Eksport
📄

Untuk simpan ke GitHub atau import semula
📊

🖨️

Cetak atau simpan sebagai PDF
📥 Import dari JSON

Import semula data dari fail JSON yang disimpan sebelum ini atau dari GitHub.

🐙 Panduan GitHub Pages
  1. Muat turun semua fail aplikasi ini (index.html)
  2. Buat repositori baru di github.com
  3. Upload fail index.html ke repositori
  4. Pergi ke Settings → Pages → pilih branch main
  5. Klik Save — URL GitHub Pages anda akan dipaparkan
  6. Eksport data sebagai JSON dan simpan di repositori sebagai data/assets.json
